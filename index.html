Hospital Ease: Desktop Patient & Appointment Management

Developed 'Hospital Ease,' a user-friendly desktop application built with Python and Tkinter. This system is designed to modernize and streamline patient data and appointment management for hospital staff, aiming to significantly save time and improve the efficiency of daily administrative operations within a healthcare setting.

Key Features:

  • Secure user login and authentication.
  • Comprehensive Patient Data Management (CRUD operations).
  • Efficient Appointment Scheduling system.
  • Intuitive GUI powered by Tkinter & data persistence with SQLite3.

Technologies: Python, Tkinter, SQLite3

Challenge: Designing an intuitive GUI with Tkinter for diverse users.

Solution: Iteratively refined UI layout, grouped functions logically, and used clear labels.

Learnings: Enhanced Python skills, GUI development with Tkinter, SQLite3 database design, user-centric design.
